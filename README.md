## About

Dr. Yan is a Research Director at the Intelligent Solution Research Center of [E Fund](https://www.efunds.com.cn/), where he leads research on **Financial Foundation Models (FinFMs)** and **Agentic Engineering** for investment research and intelligent decision-making.

Previously, he was a Principal Scientist in Huawei's [Top Minds Project](https://career.huawei.com/reccampportal/portal5/topminds.html) (2022–2024), developing AI-native products for optical networks and intelligent vehicles. He received his Ph.D. from [Tsinghua University](https://www.tsinghua.edu.cn/en/index.htm) in 2022, supervised by Prof. Xiu Li.

## Local Preview

Since the site is purely static HTML/CSS, you can preview it locally with any static server:

```bash
# Using Python
python -m http.server 8000

# Using Node.js (npx serve)
npx serve .
```

Then open `http://localhost:8000` in your browser.
